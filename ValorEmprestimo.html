Valor do Empréstimo

Atores

Funcionário

Pré-condição

O cliente fazer um empréstimo.

Fluxo de eventos principal

  1. O funcionário já deve ter solicitado "Emprestar Livro" e fornecido todos os dados corretamente.
  2. O funcionário seleciona "valor de empréstimo".
  3. O sistema pede para o funcionário verificar as informações fornecidas em "Empretar Livro".
  4. Quando o cliente fornecer as informações validas, o valor do empréstimo é disponibilizado.

Fluxo de eventos alternativos

Pós-condição

Gerar um boleto de pagamento.
