Emprestar Livro

Atores

Funcionário e cliente.

Pré-condição

Os livros a serem emprestados devem estar cadastrados, assim como os clientes devem está cadastrados.

Fluxo de eventos principal

  1. O cliente chega ao blacão com os livros que deseja.
  2. O cliente informa seu nome e entrega ao funcionário os livros.
  3. O funcionario registra o nome do cliente e inicia o empréstimo.
  4. O funcionário registra cada um dos livros.
  5. O funcionário finaliza o empréstimo, devolve os livros para o cliente e lhe informa a data de devolução e o valor de empréstimo.
  6. O cliente vai embora com os livros.

Fluxo de eventos alternativos

Pós-condição

O cliente esta com o livros em mãos, o registro para o empréstimo dos livros para o cliente já foi feito e o cliente foi informado do prazo para devolução e do valor.
