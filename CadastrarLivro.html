Cadastro de Livro

Atores

Funcionário

Pré-condição

O livro não deve está cadastrado no sistema.

Fluxo de eventos principal

  1. O funcionário seleciona "cadastar livro".
  2. O funcionario fornece o nome do livro, o nome do autor, o ISBN, a categoria, editora do livro.
  3. Se o funcionario fornecer informações de um livro que já existe, o sistema dará erro.
  4. O sistema pede para o funcionário verificar as informações novamente.
  5. Quando o funcionário fornecer as informações válidas, o cadastro é marcado como concluído.

Fluxo de eventos alternativos

Pós-condição

o livro deve está cadrastrado no sistema.
